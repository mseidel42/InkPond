## Typist Screenshot Samples

This folder contains neutral Typst sample projects for App Store / TestFlight screenshots.

Projects:

- `ieee-paper`: IEEE-style academic paper using `@preview/charged-ieee`.
- `touying-slides`: Presentation deck using `@preview/touying`.
- `product-brief`: A polished multi-page product brief with tables, callouts, and roadmap content.
- `research-poster`: A one-page landscape poster suitable for zoomed-out preview screenshots.

Notes:

- The first compile of package-based projects may download Typst packages.
- Each project uses `main.typ` as the entry file.
- Content is intentionally fictional and free of personal data.
